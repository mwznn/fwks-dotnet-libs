## Software Requirements

<details>

### Required

- [Visual Studio 2022 Latest](https://visualstudio.microsoft.com/)
- [.NET Core SDK 7](https://dotnet.microsoft.com/download/dotnet-core)
- [Chocolatey](https://chocolatey.org/)
